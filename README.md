Интерактивный веб-сайт с коктейльной картой для ресторана Grut. Современный одностраничный сайт с возможностью просмотра 22 коктейлей, их составов и скачивания изображений.

##  Описание

Проект представляет собой интерактивную коктейльную карту, созданную специально для ресторана Grut. Сайт позволяет гостям просматривать фотографии коктейлей, узнавать их состав, ингредиенты и объем, а также скачивать изображения понравившихся напитков.

### Основные возможности:

- **Просмотр коктейлей**: навигация по 22 коктейлям с фотографиями высокого качества
- **Информация о составе**: детальная информация об ингредиентах, объеме и украшении каждого коктейля
- **Интерактивная навигация**: 
  - Кнопки-стрелки для перехода между коктейлями
  - Поддержка клавиатуры (стрелки влево/вправо)
  - Свайпы на мобильных устройствах
- **Скачивание изображений**: возможность скачать фото любого коктейля
- **Адаптивный дизайн**: оптимизирован для просмотра на различных устройствах
- **Плавные анимации**: переходы между коктейлями с эффектами слайда

##  Функциональность

### Навигация

- **Стрелки влево/вправо**: переход к предыдущему/следующему коктейлю
- **Клавиатура**: 
  - `←` (стрелка влево) - предыдущий коктейль
  - `→` (стрелка вправо) - следующий коктейль
  - `Esc` - закрыть модальное окно с составом
- **Свайпы на мобильных**: горизонтальные свайпы для переключения коктейлей

### Кнопки управления

- **СОСТАВ** - открывает модальное окно с детальной информацией о коктейле (ингредиенты, объем, украшение)
- **СКАЧАТЬ** - скачивает текущее изображение коктейля

### Особенности

- **Предзагрузка изображений**: все изображения загружаются заранее для плавной работы
- **Индикатор загрузки**: отображается прогресс загрузки всех изображений
- **Счетчик**: показывает текущий номер коктейля и общее количество
- **Единая цена**: на сайте отображается единая цена для всех коктейлей - 999₽
